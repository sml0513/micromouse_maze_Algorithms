# Micromouse Maze Algorithms

농연/매연 환경에서 로봇이 안전지대까지 탈출 가능한지 검증하기 위한 CNN 기반 미로 인식 및 경로 탐색 알고리즘 실험 코드입니다.

## 프로젝트 목표

이 프로젝트는 단순한 미로 탈출 데모가 아니라, 화재 현장의 농연/매연 환경을 가정하여 로봇이 제한된 센서 정보와 CNN 기반 환경 확인 과정을 통해 안전한 대피 경로를 산출할 수 있는지 검증하는 것을 목표로 합니다.

## 주요 기능

- CNN 기반 미로/도면 확인
- 데이터 확인, 전처리, 학습, 평가, 예측
- BFS, DFS, Dijkstra, A*, 좌수법, 우수법 비교
- 무작위 장애물 추가
- 농연/매연 위험도 맵 적용
- 위험 회피 경로 탐색
- 알고리즘 성능 비교표 생성
- 미로 탈출 경로 시각화
- 연구보고서용 결론 자동 생성

## 사용 알고리즘

| 알고리즘 | 역할 |
|---|---|
| BFS | 최단 칸 수 기준 경로 탐색 |
| DFS | 깊이 우선 방식의 탐색 |
| Dijkstra | 이동 비용 기반 최적 경로 탐색 |
| A* | 휴리스틱 기반 빠른 경로 탐색 |
| 좌수법 | 왼쪽 벽을 따라가는 단순 탐색 |
| 우수법 | 오른쪽 벽을 따라가는 단순 탐색 |

## 하드웨어 연동 계획

BOM에 포함된 ESP32, ToF 거리 센서, MQ-2, MQ-7, DHT22, 불꽃센서, 모터 드라이버, DC 기어모터를 이용해 실제 로봇 구현을 목표로 합니다.

- ToF 센서: 벽 거리 측정
- MQ-2 / MQ-7: 매연 및 일산화탄소 위험도 측정
- DHT22: 온습도 측정
- 불꽃센서: 화염 감지
- ESP32: 센서값 수집 및 모터 제어
- Colab/Python 코드: CNN 분석, 경로 탐색, 결과 시각화

## 목표 결과

최종적으로 CNN 기반 환경 확인, 위험도 맵 생성, 6가지 알고리즘 비교, 안전 대피 경로 산출, 연구보고서용 표·그래프·결론 생성을 수행하는 실험 코드를 완성하는 것이 목표입니다.
