# Hardware BOM

현재 저장소는 시뮬레이션 노트북 중심 프로젝트이며 실제 하드웨어 제작이나 실제 센서 실험은 포함하지 않습니다.

## 가정 가능한 하드웨어 항목

아래 항목은 향후 실제 로봇 적용을 검토할 때의 참고용입니다.

| Category | Item | Note |
| --- | --- | --- |
| Controller | Microcontroller or SBC | 실제 제어 코드는 본 범위 밖 |
| Mobility | Differential drive chassis | 마이크로마우스/소형 주행 플랫폼 |
| Distance sensing | Wall or obstacle sensor | 미로 벽/장애물 관측용 |
| Environmental sensing | Smoke/CO/temperature sensor | 현재는 가상 CSV/synthetic 값으로 대체 |
| Power | Battery and regulator | 실제 시스템 설계 시 별도 검토 |

## Safety

실제 화재, 실제 유독가스, 실제 고온 환경 테스트는 수행하지 않습니다.
